cfg.eodHangTime = 21600000

cfg.timer = 60000

cfg.eodMngList (ARRAY)

Information required for data synchronization, array of columns : (eodMng(EOD_MNG_PROCESS_NAME), rdb(RDB_PROCESS_NAME), hbd(HDB_PROCESS_NAME)); HDB_PROCESS_NAME must have defined $dataPath value as it is used to validate data synchronisation; if array contains more than one host, sync.cfg file is required

Example

cfg.eodMngList = ((eodMng(rtr.eodMng),   rdb(rtr.rdb),   hdb(rtr.hdb)),
                  (eodMng(prod2.eodMng), rdb(prod2.rdb), hdb(prod2.hdb)),
                  (eodMng(prod3.eodMn),  rdb(prod3.rdb), hdb(prod3.hdb)))
