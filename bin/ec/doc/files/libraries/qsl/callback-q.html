callback.q

Callback library

.cb.statustable that handles collection of added callbacks
  • callback - callback (collection) name.  This is typically a standard q callback like `.z.po or `.z.pc.
  • function - a column holding the lists of names of functions to be executed
  • lastCall - the timestamp of the last callback call.
  • callCount - counts the number of times the callback was called.
