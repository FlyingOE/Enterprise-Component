.os.cpdir[dir1;dir2]

copies directory with contents.  On Linux, when dir2 points to an existing directory the actual command executed is cp -rf dir1/* dir2.  This way both on Linux and Windows the contents of dir1 is written to dir2 when dir2 already exists.

Parameter

dir1:STRINGsource dir
dir2:STRINGtarget dir
